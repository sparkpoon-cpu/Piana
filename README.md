# Piana - 安徽教招小学美术刷题 PWA

一款面向安徽教师招聘考试（小学美术）备考的渐进式 Web 应用。支持选择题和填空题，通过 DeepSeek API 联网获取题目，提供每日训练、数据分析和智能学习建议。

## ✨ 功能特性

- 📚 **三大科目覆盖** — 教育学、心理学、美术学科专业知识
- 🌐 **DeepSeek 联网搜题** — 输入知识点自动从网络获取高质量题目
- ✍️ **两种答题模式** — 选择题（ABCD点击）+ 填空题（文字输入，支持模糊匹配）
- 📊 **数据报告** — 正确率趋势图、分类统计、薄弱知识点分析
- 🤖 **AI 学习建议** — 基于错题数据生成个性化备考建议
- 📱 **PWA 离线可用** — 添加到 iPhone 主屏幕，无网络也能刷题
- 🎨 **iOS 风格精美 UI** — 毛玻璃效果、流畅动画、底部 Tab 导航

## 🚀 快速开始

### 本地运行

```bash
# 在项目目录下启动静态服务器
cd Piana
python3 -m http.server 8080

# 然后在浏览器打开
# 电脑：http://localhost:8080
# iPhone：http://<你的电脑IP>:8080
```

### 添加到 iPhone 主屏幕

1. 用 Safari 打开应用地址
2. 点击底部「分享」按钮
3. 选择「添加到主屏幕」
4. 点击「添加」即可像原生 App 一样使用

### 配置 DeepSeek API Key

1. 打开应用 → 底部「设置」Tab
2. 输入你的 [DeepSeek API Key](https://platform.deepseek.com/)
3. 密钥仅存储在本地浏览器，不会上传到任何服务器

## 📖 使用指南

### 获取题目
1. 进入「题库」Tab
2. 点击「联网获取题目」
3. 选择知识点、题目数量和类型
4. 等待 DeepSeek 返回题目（约10-20秒）
5. 也可以点击「手动添加」自行录入题目

### 每日训练
1. 进入「训练」Tab
2. 选择题量（10/20/30题）、题型（选择/填空/混合）、分类范围
3. 点击「开始答题」
4. 选择题点击选项后确认，填空题输入答案后提交
5. 完成后查看成绩单和答题回顾

### 查看报告
- 首页仪表盘：今日进度环、连续打卡、总正确率
- 报告 Tab：7天趋势图、各分类正确率、薄弱知识点 TOP5
- AI 建议：点击「AI 生成建议」获取个性化学习指导

## 🛠 技术架构

| 层级 | 技术 |
|------|------|
| 前端 | Vanilla JS + CSS（无框架依赖） |
| 存储 | IndexedDB（浏览器本地数据库） |
| API | DeepSeek Chat API（联网搜索） |
| 离线 | Service Worker + Cache API |
| PWA | Web App Manifest |

## 📝 知识点分类

预置了安徽教招小学美术的核心知识点分类：

- **教育学**（13个）：教育产生与发展、课程、教学、德育、班级管理、新课程改革等
- **心理学**（11个）：认知过程、学习理论、学习动机、学习迁移、学生心理发展等
- **美术**（7个）：美术基础知识、中国美术史、外国美术史、美术课程标准、教学设计等
